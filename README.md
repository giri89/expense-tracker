Expense Tracker 💰
📌 Description
This is a full-stack Expense Tracker application built using Java for backend logic and HTML, CSS for frontend.

It helps users track and manage their daily expenses in a simple way.

🚀 Features
Add expenses
View expense history
Simple and clean UI
🛠️ Tech Stack
Java
HTML
CSS
📂 Project Structure
Java files → Backend logic
HTML files → User interface
CSS files → Styling
